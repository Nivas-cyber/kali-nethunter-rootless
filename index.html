How to Install Kali Linux on Android Without Root – 2026

In this article, you don’t need to worry about rooting your device. We can install Kali Linux on non-rooted Android devices using Termux. Follow each step carefully and you’ll have Kali Linux running on your phone.

Step 1: Install Termux

Download Termux only from the official GitHub release. Play Store versions are outdated and broken.

⬇ Download Termux APK

Step 2: Update and Upgrade Termux

apt update && apt upgrade

This updates all Termux packages to avoid installation errors.

Step 3: Grant Storage Permission

termux-setup-storage

Allow storage access when prompted.

Step 4: Install wget Package

pkg install wget -y

wget is used to download files from the internet.

Step 5: Download NetHunter Installer

wget -O install-nethunter-termux https://offs.ec/2MceZWr

This downloads the official Kali NetHunter installer.

Step 6: Grant Executable Permission

chmod +x install-nethunter-termux

Step 7: Install NetHunter

./install-nethunter-termux

This step downloads Kali Linux (~2GB). It may take 10–20 minutes depending on your internet speed.

Step 8: Set NetHunter KeX Password

nethunter kex passwd

Step 9: Start NetHunter GUI

nethunter kex &

Connect using NetHunter KeX app from Play Store to access the Kali GUI.
